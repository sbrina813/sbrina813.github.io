Awesome packages for sublime's interface

  • tool
  • Seti_UI, seti_UX

Using package control to install package(if you do not have package control installed, please check this link), search ‘seti_UI’,install it. And using package control:install package again to search ‘seti_UX’, install.

Then, use ‘themr’ to select seti_UI to be your theme. Set Preference-Color Scheme-seti_UX.

  • themr

Themr is a UI theme selector for sublime. It allows you to quickly change your UI theme using the command palette or keyboard shortcuts(shift+command+p, type in ‘themr’).

  • Monokai color scheme
    Monokai is the color scheme I mostly used.

  • SidebarEnhancements
    Provides enhancements to the operations on Sidebar of Files and Folders for Sublime Text.
